# Orbit Finance 💰

App de finanzas personales para Android. Permite registrar ingresos y gastos, 
visualizar el estado financiero y mantener un control claro de tu dinero.

## Tecnologías

- Kotlin
- Jetpack Compose
- Firebase

## Características
